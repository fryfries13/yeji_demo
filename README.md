## 🎯 项目简介

野鸡私立大学（Wild Chicken Private University，简称 WCU）是一个**开源虚拟大学项目**，旨在讽刺教育资源垄断现象，同时为技术爱好者提供一个有趣的 .edu 邮箱实验平台。

在这个教育资源被垄断的时代，我们相信每个人都应该拥有一个属于自己的 .edu 邮箱——这不仅是一个邮箱，更是通往学生优惠、免费软件、学术资源的钥匙。

> ⚠️ **声明**：这不是真正的大学，仅供娱乐和教育目的使用。
